## About This Project
This is a simple server with express framework. When the server is accessed, it will show a message showing a user sucessfully reached the server.

## Instructions for This Project
1. Download or clone this repository into your computer.
2. Open the Powershell or Terminal and open the folder that you have donwloaded or cloned.
3. Use the command `npx start` to initiate the server.
4. Open a web browser to and go to "localhost:3000" to access the server.
